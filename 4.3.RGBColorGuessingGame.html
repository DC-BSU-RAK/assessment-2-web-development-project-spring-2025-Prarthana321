RGB Color Guessing Game

The RGB Color Guessing Game is an interactive web application designed to help users learn and test their understanding of RGB color codes. Built using HTML, CSS, and JavaScript, the game presents players with a set of colored squares and a target RGB value. Users must guess which square matches the RGB code displayed, encouraging visual recognition of color codes.

This project demonstrates skills in DOM manipulation, event handling, and dynamic styling in JavaScript. It features real-time feedback and score tracking, enhancing user engagement. The design emphasizes simplicity and clarity, making it suitable as an educational tool for beginners interested in web development and color theory.
