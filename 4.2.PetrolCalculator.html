Petrol Calculator

The Petrol Calculator is a simple, functional web app built using HTML, CSS, and JavaScript. It allows users to calculate the total cost of fuel based on user input for the number of liters and price per liter. This project demonstrates practical problem-solving through web development and enhances user experience with clean form inputs, real-time updates, and responsive styling.

It was created with a focus on simplicity and efficiency. Users just enter the fuel quantity and unit price, and the total cost is instantly displayed. This type of tool is especially useful for budgeting fuel expenses in real-time scenarios. While basic in design, it showcases my understanding of event handling, DOM manipulation, and building interactive forms in JavaScript.
